# node-blog
 Simple Blog created using Nodejs , MongoDB , Bootstrap .  
 

# Features : 
  ADD ADMIN  
  
  REMOVE ADMIN 
  
  ADMIN LOGIN 
 
  ADD POST 
  
  REMOVE POST 
  
  CREATE CATEGORY 
  
  REMOVE CATEGORY 
  
  EDIT /MANAGE ALL THINGS THAT GETS DISPLAYED ON SITE
